Three ways to declare/assign an array:

int data[] = new int[10]; // Creates blank spaces  

int[] data = {10,20,30,40,50,60,71,80,90,91}; // Has initial data but can be changed, not extended (no array can be extended)

// or

int[] data; // First initializes the array
data = new int[] {10,20,30,40,50,60,71,80,90,91}; // Declares the data. Separating the two steps from one another.
